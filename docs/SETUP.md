# Setup Guide

Welcome to A11yhood! This guide will get your local development environment up and running.

If you're setting up for the first time or on a new machine, follow this guide in order. If you're only working on one part of the stack, jump to the relevant section.

## Repositories

A11yhood is split across three repositories:

| Repository | Stack | Link |
|-----------|-------|------|
| Frontend | React 19 with TypeScript | [a11yhood/a11yhood.github.io](https://github.com/a11yhood/a11yhood.github.io) |
| Backend | Python with FastAPI and Postgres | [a11yhood/a11yhood-backend](https://github.com/a11yhood/a11yhood-backend) |
| Community | Markdown | [a11yhood/community](https://github.com/a11yhood/community) |
